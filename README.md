# NLP-Assignment1 - 662115001

## External Libraries
This project uses **NLTK (Natural Language Toolkit)** for text processing.

### Installation
To install the required library, run the following command in your terminal:

```bash
pip install nltk

```

## How to Run

* **Terminal**
  1. Open terminal and navigate to the project folder.
  2. Run the command: `python TextPreprocessing.py`
  3. Output files will appear in the folder.

* **IDE**
  1. Open the project folder in your IDE (e.g., VS Code, PyCharm).
  2. Open the Python script and click the **Run/Play** button.
  3. Output files will appear in the file explorer.
